# Documentação do Projeto

## Visão Geral do Projeto
Este projeto visa reproduzir os resultados do artigo "Epidemiologia dos Tumores Mamários Caninos no Arquipélago Canário na Espanha". O estudo investiga a incidência, preditores e implicações dos tumores mamários em cães.

## Referência
**Cita-se o artigo:**  
Silva, M. E., Pereira, A. S., & Balbinot, D. D. (2022). Epidemiology of canine mammary tumors in the Canary Archipelago, Spain. *BMC Veterinary Research*, 18, 132. [https://doi.org/10.1186/s12917-022-03363-9](https://link.springer.com/article/10.1186/s12917-022-03363-9)

## Requisitos
- Python 3.8 ou superior
- Biblioteca `pandas`
- Biblioteca `numpy`
- Biblioteca `matplotlib`
- Biblioteca `scikit-learn`
